# Casos De Usos SPLUNK

Este repositorio recopila distintos **casos de uso (Use Cases)** implementados y documentados en Splunk para la detección, investigación y monitoreo de eventos de ciberseguridad.

El objetivo es servir como una colección de consultas SPL, detecciones y escenarios comunes utilizados en un SOC para identificar actividades sospechosas, amenazas y comportamientos anómalos en diferentes entornos.
